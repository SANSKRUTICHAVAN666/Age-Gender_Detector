# Age-Gender_Detector
Simple Age and gender detector using machine learning<br>
##To simply run this model:<br>
1.Clone this repository<br>
2.Run the [gui.py](https://github.com/SANSKRUTICHAVAN666/Age-Gender_Detector/blob/main/gui.py)file<br>
3.Upload the image you want to detect<br>
4.Click on detect button<br>
5.View results<br><br>
##To run ths model yourself<br>
1.Clone this repository<br>
2.Download the dataset fro [here](https://www.kaggle.com/jangedoo/utkface-new)<br>
3.Extract the dataset to your repository folder<br>
4.Open the [Model_jupyter_notebook](https://github.com/SANSKRUTICHAVAN666/Age-Gender_Detector/blob/main/Model.ipynb)<br>
5.Run this notebook<br>
6.Run the [gui.py](https://github.com/SANSKRUTICHAVAN666/Age-Gender_Detector/blob/main/gui.py)file<br>
7.Upload the image you want to detect<br>
8.Click on detect button<br>
9.View results<br>
